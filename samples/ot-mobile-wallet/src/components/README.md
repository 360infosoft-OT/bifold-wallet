# Custom Components

This directory contains OpenText-specific UI components.

## Potential Components

- Custom header with OpenText logo
- Branded credential cards
- Custom loading indicators
- OpenText-specific icons

## Implementation Notes

Most components can be reused from Bifold core. Only create custom components
when needed for branding or unique UX requirements.
